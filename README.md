# Syntaxesia

> Where code becomes art

A post-modern art museum where code is transformed into curated art exhibitions. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

### Run the API server (FastAPI)
```bash
uvicorn api.main:app --reload
```
